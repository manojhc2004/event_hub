# event-managemen-system
This is a Event Management System built using Django. The platform allows users to browse event categories, view event details, and book their slots. Each user has their own profile page and a My Bookings section to track registered events. The system also includes an Admin Panel, where the admin can manage events, categories, and user bookings.
